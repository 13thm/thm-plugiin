# My Plugin

My Claude Code plugin.

## Features

- Custom skills
- Custom commands
- Custom agents
- Custom hooks

## Installation

1. Clone or copy this plugin to your desired location
2. Configure Claude Code to use this plugin

## Components

- `skills/` - Agent skills for specialized capabilities
- `commands/` - Slash commands for quick actions
- `agents/` - Custom agent definitions
- `hooks/` - Event hooks for automation
